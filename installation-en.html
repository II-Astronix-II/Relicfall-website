Installation Procedure

This page explains how to install, launch, and uninstall RelicFall on Windows 10 and Windows 11.

Prerequisites

No software prerequisites are required. The installer includes all necessary dependencies: Python, Pygame, and network libraries.

  • Operating system: Windows 10 or Windows 11, 64-bit
  • Processor: Intel Core i3 or equivalent
  • RAM: 2 GB
  • Disk space: 200 MB free
  • Minimum resolution: 1280 × 720
  • Local network (LAN) connection for multiplayer

Installation

Step 1 — Download the installer

Visit the project website, choose the desired version, then click the download button to download the RelicfallSetup.exe file.

Step 2 — Launch the installer

Open the folder containing RelicfallSetup.exe, then double-click the file to start the installation.

Step 3 — Allow the installation

If Windows displays a User Account Control window, click Yes to allow the installation.

Step 4 — Choose the installation folder

The wizard suggests a default installation folder. You can keep it or change it using the Browse button. Then click Next.

Step 5 — Choose the Start Menu folder

The shortcut will be placed in the RelicFall folder of the Start Menu. Leave the default value, then click Next.

Step 6 — Start the installation

The wizard shows an installation summary. Click Install to begin.

Step 7 — Finish the installation

Once the installation is complete, you can check Launch RelicFall to start the game immediately, then click Finish.

Installation result

After installation, the following items are created on your system:

— C:\Program Files\Relicfall\Relicfall.exe – main game executable

— C:\Program Files\Relicfall\unins000.exe – uninstall program

— Shortcut in the Start Menu → RelicFall folder

— Shortcut on the desktop

— Entry in Windows Add/Remove Programs

User settings and saves are stored separately in the folder: %APPDATA%\Relicfall.
      (e.g.: C:\Users\YourName\AppData\Roaming\Relicfall)

Launching the game

Solo or multiplayer host mode

Double-click the RelicFall icon on your desktop or in the Start Menu. From the main menu, choose Play locally for a solo game or Create a game to host a multiplayer session on the local network.

Multiplayer client mode

  1. Make sure the host has already started a game.
  2. Ensure both machines are connected to the same local network.
  3. Launch RelicFall on your machine.
  4. Choose Join a game.
  5. The game automatically detects the server on the local network.

Uninstall

Step 1 — Open settings

Open Windows Search, type settings, then click Open.

Step 2 — Applications

In the left menu, click Applications.

Step 3 — Installed applications

Click Installed applications.

Step 4 — Search for RelicFall

In the search bar, type RelicFall. The app appears in the list.

Step 5 — Uninstall

Click the three dots to the right of the RelicFall entry, then click Uninstall.

Step 6 — Confirm

A confirmation window appears. Click Uninstall.

Step 7 — Allow

If Windows requests permission, click Yes.

Step 8 — Confirm deletion

A dialog asks you to confirm full deletion. Click Yes.

Step 9 — Saves and settings

A second window asks whether you want to delete your saves and settings. Click Yes to delete everything or No to keep your data.
