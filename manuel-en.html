User manual

RelicFall is a top-down action-adventure game developed with Python and Pygame. The player takes on the role of an adventurer tasked with recovering lost relics in dungeons filled with monsters in order to restore balance to the world.

The game offers both a solo experience and a cooperative multiplayer mode on a local network (LAN).

Key points of the game

  • Three dungeon levels with rooms and monsters to defeat
  • A lobby between each level that lets you improve your skills
  • A merchant in the lobby who sells upgrades
  • Three skill branches: Attack, Healing, and Movement
  • Co-op multiplayer mode for up to 4 players on a local network

First launch

Double-click the RelicFall icon on your desktop or search for it in the Start Menu.

The game starts in a fixed window with a resolution of 1280 × 720.

Note: the game cannot be switched to fullscreen from the interface. Avoid manually resizing the window.

Main menu

The main menu contains four buttons:

  • Play: go to the game selection menu
  • Options: open the control configuration menu
  • Credits: display the game credits
  • Quit: close the game

Play menu

After clicking Play, three options are available:

  • Create a world: start a new game
  • Load a world: continue from a save
  • Join a game: join another player on the local network

Create a world

  1. Enter the name of your world in the central text field
  2. The name must contain between 1 and 13 characters
  3. The name must not already exist
  4. Only letters (a-z, A-Z), numbers (0-9), hyphens (-), and underscores (_) are allowed
  5. Click Create
You can keep up to 4 saved worlds at the same time.

Load a world

Select a save from the list and click Load.

A save can be deleted via the trash icon.

Deleting a save is permanent.

Multiplayer mode

Multiplayer mode works on a local network (LAN or shared Wi-Fi).

Create a game

The host must create or load a world before other players can join.

Join a game

  • Auto mode: automatic host search
  • IP mode: enter the host’s IP address

If Auto mode does not work, open a command prompt with Win + R, type cmd, and enter:

ipconfig

Then look for your IPv4 address.

All players must be connected to the same local network.

Game controls

The default controls are as follows. They can be changed in the Options menu.

Key Action
Z Move up
S Move down
Q Move left
D Move right
Space Attack
E Interact
Esc Pause menu

Game flow

A RelicFall game consists of three levels connected by lobbies.

  • Level 1: first dungeon. Objective: enter the dungeon and reach the lobby to access the other rooms
  • Lobby 1: rest area after level 1. The merchant is there
  • Level 2: second dungeon, accessible from the lobby. You must recover the first relic
  • Lobby 2: rest area after level 2. The merchant is there
  • Level 3: third and final dungeon. You must recover the last relic
  • Lobby 3: end of the game

Exploring a level

The level is made of rooms connected by doors. Doors open by eliminating all enemies in the room or by activating levers / buttons.

Enemies patrol and react to your presence. They attack when you enter their line of sight or detection zone.

In multiplayer mode, some rooms require all players to be in the same room before progressing to the next one.

At the end of a level, a relic awaits you. Interact with it to retrieve it and trigger the return to the lobby.

The lobby

The merchant is indicated by the message Press E to talk when you are close.

Doors that lead to the next level unlock as you progress.

When you first arrive in the lobby, the merchant automatically speaks to you. Press Enter to advance the dialogue, or wait for it to scroll automatically.

Combat

Press Space to attack in the direction of the mouse.

Your health points are shown at the top of the screen. If your health reaches zero, you lose and return to the previous lobby.

In multiplayer mode, a player’s death sends you back to the previous lobby.

Merchant and skill tree

Approach the merchant character and press the indicated key (E by default) to improve your skills.

Available branches

  • ATTACK in red: +10% damage per level (5 levels max)
  • HEAL in green: +10% max health per level (5 levels max)
  • MOVEMENT in blue: +10% movement speed per level (5 levels max)

Each skill level has a point cost displayed under the corresponding button. The cost increases with each unlocked level (cost = 21+current level).

Click the button for the next level to unlock it (the button after your current level is the only one active).

If you do not have enough points, the button is not clickable.

Press Esc to close the menu and save the purchased upgrades.

Pause menu

During a game, press Esc to open the pause menu. Four options are available:

  • Resume
  • Options
  • Main menu
  • Quit

Press Esc again to resume the game without clicking a button.

Options and key configuration

The Options menu is accessible from the main menu or the Pause menu.

  • Configurable actions are listed with the key currently assigned to each one
  • Click the action you want to change. The text becomes Press a key
  • Press the new key you want to assign to this action
  • Click Save to save the changes, or Return to cancel them
If your controls are misconfigured and you can no longer navigate the menus, you can manually delete the settings.json configuration file located in %APPDATA%\Relicfall. The default keys will be restored on the next launch.

Save and progress

The game automatically saves progress at the end of each level. (return to the lobby).

Saves are stored in:

%APPDATA%\Relicfall\worlds\<world_name>
You can keep up to 4 saved worlds at the same time.
There is no manual save option; if you quit in the middle of a level, you will resume at the entrance of that level on the next load.

Multiplayer summary

Host (server): launch RelicFall, go to Play, create or load a world. The game starts and waits for other players to connect.

Client (player): launch RelicFall, go to Play, and join. The game automatically detects the session on the local network and connects you. Otherwise, you can enter the host’s IP address.
