Manuel d'utilisation

Relicfall est un jeu d’action-aventure en vue de dessus développé avec Python et Pygame. Le joueur incarne un aventurier chargé de récupérer des reliques perdues dans des donjons remplis de monstres afin de rétablir l’équilibre du monde.

Le jeu propose une expérience solo ainsi qu’un mode multijoueur coopératif en réseau local (LAN).

Points clés du jeu

  • Trois niveaux de donjon avec des salles et des monstres à vaincre
  • Un lobby entre chaque niveau permettant d’améliorer ses compétences
  • Un marchand dans le lobby permettant d’acheter des améliorations
  • Trois branches de compétences : Attaque, Soin et Mouvement
  • Mode multijoueur coopératif jusqu’à 4 joueurs en réseau local

Premier lancement

Double-cliquez sur l’icône Relicfall sur le bureau ou recherchez-la dans le menu Démarrer.

Le jeu démarre dans une fenêtre fixe de résolution 1280 × 720.

Remarque : le jeu ne peut pas être mis en plein écran depuis l’interface. Évitez de redimensionner manuellement la fenêtre.

Menu principal

Le menu principal contient quatre boutons :

  • Jouer : accéder au menu de sélection de partie
  • Options : ouvre le menu de configuration des touches
  • Crédits : afficher les crédits du jeu
  • Quitter : fermer le jeu

Menu Jouer

Après avoir cliqué sur Jouer, trois possibilités sont disponibles :

  • Créer un monde : créer une nouvelle partie
  • Charger un monde : reprendre une sauvegarde
  • Rejoindre une partie : rejoindre un autre joueur en réseau local

Créer un monde

  1. Saisissez le nom de votre monde dans le champ de texte central
  2. Le nom doit contenir entre 1 et 13 caractères
  3. Le nom ne doit pas déjà exister
  4. Seules les lettres (a-z, A-Z), chiffres (0-9), tirets (-) et underscores (_) sont autorisés
  5. Cliquez sur Créer
Vous pouvez posséder jusqu’à 4 mondes sauvegardés simultanément.

Charger un monde

Sélectionnez une sauvegarde dans la liste puis cliquez sur Charger.

Une sauvegarde peut être supprimée via l’icône corbeille.

La suppression d’une sauvegarde est définitive.

Mode multijoueur

Le mode multijoueur fonctionne en réseau local (LAN ou Wi-Fi partagé).

Créer une partie

L’hôte doit créer ou charger un monde avant que les autres joueurs puissent rejoindre.

Rejoindre une partie

  • Mode Auto : recherche automatique de l’hôte
  • Mode IP : saisie de l’adresse IP de l’hôte

Si le mode automatique ne fonctionne pas, ouvrez une invite de commande avec Win + R, tapez cmd, puis écrivez :

ipconfig

Recherchez ensuite votre IP sur la ligne Adresse IPv4.

Tous les joueurs doivent être connectés au même réseau local.

Commandes du jeu

Les commandes par défaut sont les suivantes. Elles peuvent être modifiées dans le menu Options.

Touche Action
Z Déplacement vers le haut
S Déplacement vers le bas
Q Déplacement vers la gauche
D Déplacement vers la droite
Espace Attaquer
E Interagir
Échap Menu pause

Déroulement d'une partie

Une partie Relicfall se compose de trois niveaux reliés par des lobbys.

  • Niveau 1 : premier donjon. Objectif : pénétrer dans le donjon afin de rejoindre le lobby pour accéder aux autres salles
  • Lobby 1 : zone de repos après le niveau 1. Le marchand y est présent
  • Niveau 2 : deuxième donjon, accessible depuis le lobby. Vous devez récupérer la première relique
  • Lobby 2 : zone de repos après le niveau 2. Le marchand y est présent
  • Niveau 3 : troisième et dernier donjon. Vous devez récupérer la dernière relique
  • Lobby 3 : fin du jeu

Exploration d'un niveau

Le niveau est composé de salles reliées entre elles par des portes. Les portes s’ouvrent en éliminant tous les ennemis présents dans la salle ou en activant des leviers / bouttons.

Des ennemis patrouillent et réagissent à votre présence. Ils vous attaquent lorsque vous entrez dans leur champ de vision ou leur zone de détection.

En mode multijoueur, certaines salles nécessitent que tous les joueurs soient présents dans la même salle pour progresser vers la salle suivante.

À la fin d'un niveau, une relique vous attend. Interagissez avec elle pour la récupérer et déclencher le retour vers le lobby.

Le lobby

Le marchand est indiqué par le message « Appuie sur E pour parler » lorsque vous êtes à proximité.

Des portes donnant accès au niveau suivant (elles se déverrouillent au fil de l’avancement).

Lors de votre première arrivée dans le lobby, le marchand vous adresse automatiquement la parole. Appuyez sur la touche entrer pour avancer dans le dialogue, ou attendez qu’il défile seul.

Combat

Appuyez sur Espace pour attaquer dans la direction de la souris.

Vos points de vie sont affichés en haut de l’écran. Si vos points de vie tombent à zéro, vous perdez et retournez au lobby précédent.

En mode multijoueur, la mort d’un joueur vous fait retourner au lobby précédent.

Marchand et arbre de compétences

Approchez-vous du marchand et appuyez sur la touche indiquée (E par défaut) pour ouvrir le menu d’amélioration des compétences.

Branches disponibles

  • ATTACK en rouge : +10% de dégâts infligés par niveau (5 niveaux max)
  • HEAL en vert : +10% de points de vie max par niveau (5 niveaux max)
  • MOUVEMENT en bleu : +10% de vitesse de déplacement par niveau (5 niveaux max)

Chaque niveau de compétence a un coût en points affiché sous le bouton correspondant. Le coût augmente à chaque niveau débloqué (coût = 21+niveau actuel).

Cliquez sur le bouton du prochain niveau à débloquer pour l’acheter (le bouton suivant votre niveau actuel est le seul actif).

Si vous n’avez pas assez de points, le bouton n’est pas cliquable.

Appuyez sur Échap pour fermer le menu et sauvegarder les améliorations achetées.

Menu Pause

Pendant une partie, appuyez sur Échap pour ouvrir le menu pause. Quatre options sont disponibles :

  • Reprendre
  • Options
  • Menu principal
  • Quitter

Appuyez à nouveau sur Échap pour reprendre la partie sans cliquer sur un bouton.

Options et configuration des touches

Le menu Options est accessible depuis le menu principal ou depuis le menu Pause.

  • Les actions configurables sont listées avec la touche actuellement associée à chacune
  • Cliquez sur l’action que vous souhaitez modifier. Le texte passe à Appuyez sur une touche
  • Appuyez sur la nouvelle touche souhaitée pour l’associer à cette action
  • Cliquez sur Sauvegarder pour enregistrer les modifications, ou sur Retour pour les annuler
Si vos touches sont mal configurées et que vous ne parvenez plus à naviguer dans les menus, vous pouvez supprimer manuellement le fichier de configuration settings.json situé dans %APPDATA%\Relicfall. Les touches par défaut seront restaurées au prochain lancement.

Sauvegarde et progression

Le jeu sauvegarde automatiquement la progression à la fin de chaque niveau. (retour dans le lobby).

Les sauvegardes sont stockées dans :

%APPDATA%\Relicfall\worlds\<nom_du_monde>
Vous pouvez posséder jusqu’à 4 mondes sauvegardés simultanément.
Il n’existe pas de sauvegarde manuelle; si vous quittez en plein milieu d’un niveau, vous reprendrez à l’entrée de ce niveau lors du prochain chargement.

Résumé du mode multijoueur

Hôte (serveur) : Lancer Relicfall, Jouer, Créer ou Charger un monde. Le jeu démarre et attend les connexions des autres joueurs.

Client (joueur) : Lancer Relicfall, Jouer, Rejoindre . Le jeu détecte automatiquement la session sur le réseau local et vous connecte. Sinon, vous avez la possibilité de rentrer l’IP de l’hôte.
